# Konex Desktop

Application desktop officielle pour [Konex](https://konex.lol) - Chat & Voice.

## Installation

Télécharge la dernière version depuis les [Releases](https://github.com/mybloodallovertheroom/konex-desktop/releases).

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run dist        # toutes plateformes
npm run dist:win    # Windows
npm run dist:linux  # Linux
```

## Technologies

- Electron
- React + TypeScript
- Vite
- Tailwind CSS
- electron-updater
